# [Zombie Invaders game](https://t73liu.github.io/zombie-invaders/)

## Controls

- Arrow Keys to move the Player
- Space to Fire

## Setup

1. Clone the repository.
2. Open index.html using Firefox

## Tested Browsers

- Chrome 58+
- Firefox 53+
